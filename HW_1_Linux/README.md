## Домашнее задание №1 

### **Тема:** Linux

---

### **Постановка задачи:**
```
Нужно написать shell файл: 
Который принимает на вход три параметра START|STOP|STATUS. 
START запускает его в фоне и выдает PID процесса, 
STATUS выдает состояние - запущен/нет, 
STOP - останавливает PID
Сам shell мониторит утилизацию дискового пространства, количество свободных inode. Выводит информацию в виде csv файла. Имя файла должно содержать timestamp запуска + дату за которую мониторинг. Предусмотреть создание нового файла при переходе через сутки
```

---

### **Использование и описание решения:**

Для запуска скрипта выполнить команду: `./run.sh START`

Для показа состояния и остановки выполнить команды: `./run.sh STATUS` и `./run.sh STOP` соответственно.

Данные будут сохраняться в csv файле в созданной скриптом директории `disk_monitor` с интервалом в 30 секунд. Установить путь и частоту можно в начале файла `run.sh` в переменных `LOG_DIR` и `INTERVAL`

В той же директории в момент выполнения будет создан файл `disk_monitor.pid` содержащий информацию о текущем pid запущенного процесса

---

### **Описание атрибутов сохраняемых данных:**

| Атрибут       |    Описание     |
|----------------|-----------------|
| Timestamp      | Дата и время, когда был произведен сбор данных |
| Filesystem     | Имя файловой системы или раздела, для которого собирается информация | 
| Disk_Size      | Общий объем дискового пространства, доступного на данный момент |
| Disk_Used      | Объем дискового пространства, который уже занят |
| Disk_Usage(%)  | Процент использованного пространства относительно общего объема |
| IUsed          | Количество занятых inode |
| IFree          | Количество свободных inode, которые еще могут быть использованы для создания новых файлов и директорий |
|


**Примечание:** IFree (количество свободных inode) - основания цель сбора данных, согласно постановке задачи
